# 🤖 WhatsApp AI Chatbot

A full-stack WhatsApp chatbot built with:
- Next.js
- Supabase
- WhatsApp Cloud API
- OpenRouter (AI)

---

## 🚀 Features

- 💬 Real-time WhatsApp chat
- 🤖 AI auto-replies (Agent Mode)
- 👨 Human takeover mode
- 📊 Dashboard to manage conversations
- 🔗 Webhook integration (Meta API)

---

## 🛠️ Tech Stack

- Next.js (Frontend + API)
- Supabase (Database)
- WhatsApp Cloud API
- OpenRouter AI
